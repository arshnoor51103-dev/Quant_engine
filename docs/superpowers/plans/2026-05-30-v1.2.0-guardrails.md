```python
def apply_drawdown_halt(
    cards: list[TradeCard],
    current_drawdown: float,
    ceiling: float,
    enabled: bool = True,
) -> tuple[list[TradeCard], bool]:
    """
    Soft-halt new risk at the drawdown ceiling.

    When ``enabled`` and ``current_drawdown >= ceiling``, every BUY card is
    converted to a SKIP card with gate_status DRAWDOWN_HALT. SELL / HOLD / WARN
    cards are left untouched (risk reduction and rebalancing still flow).

    Args:
        cards:            output of generate_trade_cards (mutated in place)
        current_drawdown: positive magnitude of current portfolio drawdown (e.g. 0.22)
        ceiling:          risk.max_drawdown from config (e.g. 0.20)
        enabled:          risk.drawdown_halt_enabled toggle

    Returns:
        (cards, halted) — halted True if the ceiling was breached and halting is on.
    """
    if not enabled or current_drawdown < ceiling:
        return cards, False
    for card in cards:
        if card.action == "BUY":
            card.action = "SKIP"
            card.gate_status = GateStatus.DRAWDOWN_HALT
            card.gate_reason = (
                f"drawdown {current_drawdown:.1%} >= {ceiling:.0%} ceiling — BUYs halted"
            )
    return cards, True
```

```python
    def ticker_metadata(self, ticker: str) -> dict:
        """
        Extract a ticker-specific metadata slice for per-row persistence.

        Contract (binding on all Signal implementations):
        - Per-ticker keys (registered in _PER_TICKER_KEYS): value is a
          {ticker: x} dict. Extract value[ticker], rename via
          _PER_TICKER_KEY_MAP. Omitted if ticker absent.
        - List values: dropped from per-ticker rows (e.g. skipped_tickers) —
          run-level data does not belong on every row. (F20)
        - Everything else (scalars AND structural dicts like regime_weights):
          broadcast verbatim. Structural dicts are preserved, not dropped. (F6)

        Returns an empty dict if metadata is None.
        """
        if not self.metadata:
            return {}
        out: dict = {}
        for key, value in self.metadata.items():
            if key in _PER_TICKER_KEYS:
                if isinstance(value, dict) and ticker in value:
                    out[_PER_TICKER_KEY_MAP.get(key, key)] = value[ticker]
            elif isinstance(value, list):
                continue  # F20: do not broadcast run-level lists per row
            else:
                out[key] = value  # scalar or structural dict — broadcast verbatim
        return out
```
